## 🎯 Project Overview

This is my Personal AI Employee project.
It can automatically handle tasks like a digital assistant.

## ⚙️ Features

* Detect new tasks
* Create plans
* Set priority (HIGH / NORMAL)
* Suggest replies
* Ask for human approval
* Complete tasks after approval
* Update dashboard

## 📂 Folder Structure

* Inbox
* Needs_Action
* Plans
* Done
* approval_pending
* approval_done

## 🚀 How it Works

1. Task add in Inbox
2. Moves to Needs_Action
3. AI creates plan
4. Approval required
5. User approves
6. Task completed
